## Research-session test plan

### User flow

1. Open production site (or staging) in a clean browser profile with analytics instrumentation enabled.
2. Navigate to `/topic/rag` or `/topic/statistics-for-ml` (or search `what is rag` / `statistics for machine learning` and land on topic hub).
3. Scroll **Compare explanations** — open at least two contrasting rows (beginner vs technical if available).
4. Save one moment to the local library (saved clips).
5. Copy citation (plain or markdown) from the citation-ready moment card.
6. Click a **related topic** chip or internal link to a neighbor hub (e.g. transformers, probability).
7. Open **Saved** / library and revisit the saved clip (same tab session).
